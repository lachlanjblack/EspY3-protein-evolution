# EspY3-protein-evolution

This repository contains data and analysis used in my thesis 'Phylogentic and structural analysis of the enterohaemmorhagic Escherichia coli type III secretion system effector EspY3'

## Contents
- `data_and_analysis` – all the data and analysis used in the final thesis 
- `gallery/` - collection of images that may be used in the final thesis
- `tools/` – collection of some of the tools used in the final thesis or not
- `archive/` – collection of analysis and data that were not used in the final thesis
